# Чек-лист приемки (критерии из ТЗ)

## Функциональность
- Загрузка фото с сайта:
  - drag-and-drop и множественная загрузка
  - сохранение файла и метаданных (дата загрузки, категория, теги)
  - поиск/фильтрация по `q`, `category`, `tags`, `file_type`
  - пагинация
- Загрузка фото из Telegram:
  - одиночные фото и альбомы (`media_group_id`)
  - сохранение в общий архив
  - поиск и выдача результатов
  - экспорт `zip`

## Роли/безопасность
- `admin` может создавать категории/теги и удалять фото
- `user` может загружать и пользоваться поиском/галереей
- Для бота работает авторизация через `BOT_API_KEY` и заголовок `X-Bot-Token`

## Производительность
- При данных масштаба >= 1000 фото:
  - сценарий upload -> index -> search выполняется целевым образом (в ТЗ: <= 5 сек)
  - бот отвечает на поиск/выдачу в разумных пределах (в ТЗ: 2-3 сек на типовые запросы)

## Кросс-устройство
- Сайт корректно выглядит и работает на mobile/desktop (минимум 3 устройства или браузер-эмуляции).
