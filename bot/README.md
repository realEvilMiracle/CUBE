# Telegram bot (aiogram)

Бот загружает фото в общий архив через backend API и позволяет делать поиск/экспорт.

## Переменные окружения
- `TELEGRAM_BOT_TOKEN` — токен бота Telegram
- `BOT_API_KEY` — токен для авторизации бота в backend (заголовок `X-Bot-Token`)
- `BACKEND_BASE_URL` — базовый URL backend (например, `http://localhost:8000`)

## Команды
- `/start` — справка
- Отправьте фото в чат — бот сохранит фото в backend
- `/search <ключевые слова>` — поиск и отправка первых результатов
- `/export <ключевые слова>` — выгрузка zip по выборке

## Метаданные из подписи
- Теги: `#tag1 #tag2`
- Категория: `category: природа` (или `Категория: природа`)
